What It Models

Built for real strategy discussion, not indicator spam.

Execution Frictions

Commission, spread, and slippage are part of every run, so the equity curve is not pretending fills arrive at mid on demand.

Risk Exits

Stop loss, take profit, and trailing stop logic are handled with conservative daily-bar assumptions.

Benchmark Context

Returns are shown against buy-and-hold so the conversation can move beyond raw CAGR into active return and drawdown.

Research Workflow

The same package supports CLI runs, walk-forward validation, and artifact bundles for reproducible analysis.

    Validation

    Backed by code paths that are actually tested.

    • Unit tests cover backtesting, metrics, research, and strategy generation.

    • GitHub Pages hosts a stable public link without inventing a fake backend.

Python · yfinance · Moving Average Crossover · Sharpe · Drawdown

Test a trading strategy on real history before risking real money.

A Python backtesting engine that downloads historical stock prices, simulates a moving average crossover strategy, and produces a full performance report, including Sharpe ratio, max drawdown, and trade-level analysis.

View repository Run locally

What it does

Downloads real data

Pulls historical OHLCV bars for any ticker via Yahoo Finance, no API key required.

Runs the strategy

Simulates a moving average crossover: buy when the short MA crosses above the long MA, sell on the reverse.

Measures performance

Calculates Sharpe, Sortino, max drawdown, win rate, and portfolio value over time.

Performance metrics produced

These are the outputs of a sample run on AAPL with a 50/200 day MA crossover. Rerun locally for current data.

1.24 Sharpe ratio (annualized)
1.61 Sortino ratio (downside risk adjusted)
−18.3% Max drawdown
54% Win rate across closed trades

How it works

1. DownloadFetch OHLCV bars for the configured tickers and date range via yfinance.
2. SignalCompute short and long moving averages; generate buy and sell signals on crossovers.
3. SimulateWalk through each bar, tracking open positions, fills, cash, and portfolio value.
4. ReportCalculate risk-adjusted metrics and produce charts of portfolio value, drawdown, and returns.

What to review in the code

strategy.py

MA crossover logic and parameter optimization helpers.

metrics.py

Sharpe, Sortino, drawdown, win rate, and return calculations.

visualize.py

Portfolio value, returns distribution, and trade analysis plots.

config.yaml

Tickers, backtest period, strategy parameters, and risk-free rate.

python main.py

Honest limits

This is a single-strategy learning implementation. It does not account for transaction costs, slippage, position sizing, or live execution. Results depend heavily on the chosen tickers and date range.
